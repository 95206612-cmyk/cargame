# static/textures/ui-sprite/ — UI & HUD Sprites

## Naming Convention
- Lowercase, dash-separated: `icon-nitro.png`, `minimap-marker.png`, `rank-badge-gold.png`

## Requirements
- Power-of-two dimensions
- RGBA PNG with transparency
- Designed for `@2x` (retina) — UI scales down for lower DPIs

## Sprite Categories
| Prefix       | Usage                    |
|-------------|--------------------------|
| icon-       | HUD action icons         |
| marker-     | Minimap markers          |
| badge-      | Rank/achievement badges  |
| gauge-      | Speedometer/tachometer   |
| panel-      | UI panel backgrounds     |

## Supported Formats
| Format | Status |
|--------|--------|
| `.png` | primary |
